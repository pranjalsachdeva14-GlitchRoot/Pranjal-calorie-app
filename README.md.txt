# Weather Data Visualizer

This project analyzes weather data using Python for an academic assignment.

## Features
- Data cleaning using Pandas
- Statistical analysis using NumPy
- Visualizations using Matplotlib
- Grouping and aggregation
- Exported cleaned CSV + PNG charts

## Files
- weather_data.csv
- visualize_weather.py
- cleaned_weather_data.csv
- PNG images
- report.md (full analysis)
